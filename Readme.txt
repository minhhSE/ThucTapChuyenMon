Các bước cài đặt và build chương trình

Bước 1: Tải expo trên điện thoại
Bước 2: Download source code trên github
Bước 3: Vào Command Prompt
Bước 4: Gõ lệnh cd với đường lẫn source code được download về(Ví dụ cd C:\ThucTapChuyenMon)
Bước 5: Gõ lệnh code . project sẽ được mở lên trong visual studio code
Bước 6: Click vào Terminal(Thanh lệnh góc trái trên cùng) sau đó chọn New Terminal hoặc gõ phím tắt Ctrl+Shift+` để mở Terminal
Bước 7: Gõ lệnh npm start trong Terminal để chạy bài
Bước 8: Sau khi gõ lệnh npm start thì sẽ xuất hiện mã QR code
Bước 9: Mở expo trên điện thoại vào phần Scan QR code để quét mã QR code
Bước 10: Ứng dụng sẽ hiện lên trên điện thoại và thầy cô có thể trải nghiệm
